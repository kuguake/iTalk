# iTalk - A web IM

iTalk is a web IM based on NodeJS and Faye.

## Dependencies:

> Node: 0.10.33

> NPM: 1.4.28

> body-parser: 1.12.2

> ejs:1.0.0

> express: 4.1.1

> faye: 1.1.1

> mongodb: 1.4.35

> mongoskin: 1.4.13


## How to start?

* Step 1: Install modules
```sh
npm install
```

* Step 2: Start server

```sh
npm start
```

    Open It on web browser ：http://localhost:8000/chat?from=YOUR_USERNAME
